# GitNova Demo Walkthrough Script (5 Minutes)

## Introduction (0:00 - 0:30)

**What to click/navigate to:** Start on the landing page (`/`).

**What to highlight with cursor:**
- Hover over the hero section title "Learn Git the way it should be taught."
- Point to the "Start Learning Free" CTA button.
- Briefly move over the animated Git branch SVG.

**Key talking points:**
- GitNova is a gamified, interactive Git learning platform designed for beginners and intermediate developers.
- Target audience: CS students, self-taught developers, bootcamp graduates, and anyone wanting to master Git fundamentals.
- Key features: Interactive lessons, simulated terminal, XP/leveling system, boss challenges, and progress tracking.
- Free to play with 50+ levels across 5 language tracks.

**Expected visual result:** Animated landing page with particle effects, gradient text, and a responsive hero section.

---

## Landing Page (0:30 - 1:00)

**What to click/navigate to:** Scroll down the landing page to show sections.

**What to highlight with cursor:**
- Scroll to the "Choose Your Path" language tracks section.
- Hover over the Git track card (highlighted with "POPULAR" badge).
- Scroll to the "Why developers love it" feature cards.
- Point to the feature icons (Interactive Lessons, Real Terminal, Level Up & Earn XP, Boss Challenges).

**Key talking points:**
- Five learning tracks: Git, Python, C, C++, Java — each with unique levels and challenges.
- The platform uses a modern tech stack: React, Node.js, TypeScript, and is fully browser-based.
- Testimonials from early testers and students show the platform's effectiveness.
- The CTA at the bottom reinforces the free, accessible nature of the tool.

**Expected visual result:** Smooth scrolling with parallax effects, animated feature cards, and a testimonial carousel.

## Registration (1:00 - 1:30)

**What to click/navigate to:** Click "Sign In" in the header, then click "Create account" link.

**What to highlight with cursor:**
- Fill out the form fields: Full Name, Username, Email, Password, Confirm Password.
- Show validation errors (e.g., invalid email, short password).
- Click through the 3-step registration process: Info → Track → Avatar.

**Key talking points:**
- Simple, 3-step registration process with progress bar.
- Form validation ensures data quality (email format, password length, username uniqueness).
- Users can choose their preferred language track and avatar during registration.
- Social login options (GitHub, Google, Microsoft, LinkedIn) for quick onboarding.

**Expected visual result:** Split-screen layout with animated growing branch SVG on the left, form with floating labels on the right.

## Level Selection (2:20 - 2:50)

**What to highlight with cursor:**
- Hover over level nodes on the map (showing tooltips with level info).
- Point to the different node colors (concept=green, practice=blue, challenge=orange, boss=gold).
- Show the lock state for uncompleted levels.
- Click on an unlocked level node.

**Key talking points:**
- Visual level map with interconnected nodes showing progression path.
- Color-coded categories help users understand lesson types.
- Boss levels (hexagonal nodes) combine multiple concepts.
- Tooltips show estimated time and XP reward for each level.

**Expected visual result:** SVG-based level map with animated path drawing, glowing current level indicator, and hover tooltips.

**Key talking points:**
- Three-panel layout mirrors professional IDE experience.
